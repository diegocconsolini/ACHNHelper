B: Toolbar + Grid + Bottom Bar

Horizontal toolbar on top (species/color picker), grid takes maximum space, collapsible info bar at bottom. Grid is as large as possible. Click a cell to show details in the bottom bar.

C: Grid + Tabbed Sidebar (Breed / Simulate / Save)

Palette strip above grid, tabbed right panel for breeding info, simulation controls, and saved layouts. Breeding tab shows selected cell details and adjacent pairs. Simulate tab runs day-by-day evolution. Combines all features without cluttering the grid.
